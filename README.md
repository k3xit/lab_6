# Поморцев Аким КМБ-1 Лабораторная №6

# Задание 1

## Задача 1

### Текст задачи

Разработать класс с одним строковым полем. Создать конструктор копирования. Разработать
метод, создающий строку из первого и последнего символов поля. Перегрузить метод ToString()
для формирования строки из полей класса. Реализовать дочерний класс (его содержание
предложить самостоятельно и описать в решении: какой содержательный смысл имеют поля;
реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать все
конструкторы и другие методы базового и дочернего классов.

### Алгоритм решения

Создаём базовый класс с полем бренда и тремя конструкторами (по умолчанию, с параметрами и копирования). Добавляем свойство с проверкой строки на пустоту, метод для получения первого и последнего символа названия и переопределённый ToString. Разрабатываем дочерний класс, наследуем базовый и добавляем поля модели, года выпуска и мощности. Реализуем аналогичные конструкторы, передавая бренд в базовый класс через base. Пишем метод проверки повышенного налога (если мощность больше 160 л.с.) и метод формирования полного описания автомобиля. В программе последовательно создаём машины разными конструкторами, выводим их на экран, проверяем налог и получаем символы бренда. Переопределённый ToString автоматически выводит расширенные данные дочернего класса. Программа завершает работу.

### Алгоритм решения

Создаётся класс Time с приватными полями часов и минут типа byte. Реализуются публичные свойства с валидацией диапазонов (0–23 и 0–59) и автоматическим перерасчётом при превышении 59 минут. Описываются четыре конструктора: по умолчанию, с указанием часов и минут, из общего количества минут (uint) и конструктор копирования. Перегружается метод ToString() для вывода времени в формате ЧЧ:ММ. Добавляется метод AddMinutes(), увеличивающий время на заданное количество минут с учётом перехода через сутки.

Перегружаются унарные операторы ++ и -- (добавление/вычитание одной минуты с циклическим переходом через 23:59 <-> 00:00), бинарные операторы + и - для сложения и вычитания минут (поддержка лево- и правосторонних операций с типом uint). Реализуются операторы приведения типов: явное приведение к byte возвращает количество часов, неявное к bool возвращает true, если время не равно 00:00, и false в противном случае.

В основной программе последовательно создаются объекты всеми конструкторами, выводятся на экран и тестируются все методы и перегруженные операторы. Реализован ввод с клавиатуры через статический метод ReadFromConsole(), который поочередно запрашивает часы и минуты, проверяет корректность ввода через byte.TryParse и границы диапазонов, при ошибке выводит сообщение и повторяет запрос. Итоговые значения всех операций выводятся в консоль. Программа завершает работу.
