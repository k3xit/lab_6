# Задание 1

## Задача 1

### Текст задачи

Разработать класс с одним строковым полем. Создать конструктор копирования. Разработать
метод, создающий строку из первого и последнего символов поля. Перегрузить метод ToString()
для формирования строки из полей класса. Реализовать дочерний класс (его содержание
предложить самостоятельно и описать в решении: какой содержательный смысл имеют поля;
реализовать конструкторы; предложить и реализовать 2-3 метода). Протестировать все
конструкторы и другие методы базового и дочернего классов.

### Алгоритм решения

Создаём базовый класс с полем бренда и тремя конструкторами (по умолчанию, с параметрами и копирования). Добавляем свойство с проверкой строки на пустоту, метод для получения первого и последнего символа названия и переопределённый ToString. Разрабатываем дочерний класс, наследуем базовый и добавляем поля модели, года выпуска и мощности. Реализуем аналогичные конструкторы, передавая бренд в базовый класс через base. Пишем метод проверки повышенного налога (если мощность больше 160 л.с.) и метод формирования полного описания автомобиля. В программе последовательно создаём машины разными конструкторами, выводим их на экран, проверяем налог и получаем символы бренда. Переопределённый ToString автоматически выводит расширенные данные дочернего класса. Программа завершает работу.
